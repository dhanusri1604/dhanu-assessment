# Product Card - Stationery

## What I Did
Created 3 product cards for stationery items (Pen, Pencil, Eraser) using HTML and CSS. Each card contains an image, product name, description, price, and a Buy Now button.

## HTML Structure
```html
<div class="product-card">
    <img src="pen.jpg" alt="Pen">
    <h3>Classic Ball Pen</h3>
    <p>Smooth writing, blue ink, comfortable grip.</p>
    <p class="price">₹10</p>
    <button>Buy Now</button>
</div>
```

## CSS Styling
```css
.product-card {
    width: 200px;
    border: 1px solid #ccc;
    border-radius: 10px;
    padding: 15px;
}
```

## Concepts Used
- `<div>` for card container
- `<img>` for product image
- `class` attribute for CSS styling
- `<button>` for Buy Now action
